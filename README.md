[Colors Game](https://mahrokhnz.github.io/colors-game/)

## Run Locally

Clone the project

```bash
  git clone git@github.com:mahrokhnz/colors-game.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

Build project

```bash
  npm run build
```
